# Egalitarian-Gradient-Descent
In main-results folder, there are codes to reproduce main results for EGD, EGD+RSVD, Column Normalization, and SGD for modulo addition, multiplication, and parity tasks. 

In grokfast-comparison folder, there are codes to reproduce results of comparison between grokfast and EGD. sh files showing how to run these codes are provided. The main body of code obtained from official repo for "Grokfast: Accelerated Grokking by Amplifying Slow Gradients" cloned from https://github.com/ironjr/grokfast on Sept. 10, 2025.
